PROJECTS PAGE

This page consists of the projects done during the year. Click the buttons below to explore the projects from each academic year.


ANIMATION PROJECT

Using Python and the knowledge of loops, an animation was made by creating different shapes and moving them at different speeds along the x and y axes. The final product was meant to tell a story, having a motif and a thematic message.

AVATAR PROJECT

Using Python, create an avatar using different shapes in different places, overlapping them and utilizing color to create a final cohesive product. The end result is a program producing a graphic resembling an icon, with some kind of figure on the inside.

EMOTICON GAME

Using Python, address an issue pertaining to either recycling, the ocean, or the environment by creating a game about it. The final product should be a game of any type that is well-thought out and designed while being applied to the original theme.

EARSKETCH RINGTONE

Using Python and Earsketch, create loops of soundtracks into a coherent piece someone may select when choosing their ringtone. The overall product is meant to be a cohesive musical piece using loops, variables, and functions.

EARSKETCH SONG

Using Python to create a musical piece, participate in a competition regarding music production in Earsketch. The final product should be a creative song where coding techniques are used to optimize efficiency.

P1 COLORED DOTS

Using a combination of web development knowledge as well as Python, the end goal was to create a website where when the mouse was hovered over a specific box, colored dots would begin to flash on the screen.

PING PONG GAME

Using Python, create a functioning ping pong game. The end product should have the main components of a ping pong game that work together coherently to provide a pleasant user gameplay experience.

INFY APP CHALLENGE

Tackling a social issue of students' choice, create an idea for an app that can be utilized to lessen or solve the social issue. The final product should be a coherent app, finished with a presentation, showcasing it and its potential.

GHOST SCHOOL WEBSITE

After completing introduction to programming and psuedocode Ghost School, participate in the SIA Final Project and create a website where students are able to portray it. They will gain a better understanding of code in general.

LOVED PETS WEBSITE

Taking on a fake client's request to create a website, students will emulate a real-world experience with creating websites for specific organizations who need to cater to a targeted audience's needs.

VISION BOARD 2025

Reflecting on the previous year and what is wanted for the future, students will gather images, create articulate and measureable goals, as well as craft a vision board pertaining to the future they want to achieve in the upcoming years.

CHARACTER QUIZ

Learning new tool and programming language jQuery, students will combine new efforts with old information to create a website the user can interact with as a quiz, having their choices impact the outcome they receive.

CONGRESSIONAL APP CHALLENGE

In an app creation competition, students will create an application of their choice to solve an issue they witness within their lives. They will then create a video depicting their app's purpose, target audience, and functionality.
